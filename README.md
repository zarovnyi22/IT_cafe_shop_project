# Система обліку кав'ярні

Легка демонстраційна реалізація POS/складського обліку для кав'ярні з двома ролями (Barista та Admin). Сервер працює на стандартних бібліотеках Python і SQLite, тому не потребує додаткових залежностей.

## Запуск
1. Запустіть сервер: `python app.py`
2. Відкрийте інтерфейс: http://localhost:8000

Під час першого запуску база даних створюється автоматично на основі схеми `coffee_shop_db.sql` та наповнюється початковими даними.

### Облікові записи
- Адмін: `+380991234567` / `admin123`
- Бариста: `+380997654321` / `barista123`

## Ключовий функціонал
- Авторизація з ролями та токенами.
- Створення замовлення з автоматичним списанням інгредієнтів.
- Перегляд складських залишків та індикація критичних запасів.
- CRUD для інгредієнтів і продуктів (роль Admin).
- Звіт продажів за день/тиждень/місяць.
